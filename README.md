# NYC Jobs Search 📱

A modern Android application built with **Jetpack Compose** that allows users to browse, search, and save NYC Government job listings. The app focuses on clean architecture, smooth UI/UX, and efficient data handling using modern Android development practices.

---

## ✨ Features

- 🔍 **Job Search** – Search NYC government job listings with real-time filtering  
- ⭐ **Favorites System** – Save and manage your preferred job postings  
- 📄 **Job Details View** – View detailed information about each job  
- 🔄 **Multi-Screen Navigation** – Smooth navigation between screens  
- 📶 **Retrofit API Integration** – Fetches live job data from a REST API  
- 🧠 **MVVM Architecture** – Structured and scalable code using ViewModel + StateFlow  
- ⚡ **Loading & Error States** – Proper UI feedback during network operations
